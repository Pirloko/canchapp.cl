# landingPage-Canchapp

Landing page pública de Canchapp (B2B — dueños de centros deportivos), lista para abrir en el navegador.

## Archivos
- `landingPage-Canchapp.dc.html` — la página completa (abrir directo en el navegador)
- `image-slot.js` — componente de huecos de imagen (arrastra una imagen encima para llenarlos)
- `support.js` — runtime necesario; debe estar junto al HTML

## Pendientes antes de publicar
1. **Foto del hero**: arrastra una foto real de tus canchas al hueco del hero.
2. **Capturas**: 3 capturas de la app en "Cómo funciona".
3. **Testimonios/logos**: reemplaza la sección "Centros que ya usan Canchapp" (o escóndela).
4. **Formulario**: conecta el formulario final a Calendly, correo o WhatsApp.
5. **Logo**: reemplaza el wordmark cuando tengas logo definitivo.
